# Contributing

ReferralKit is still being extracted and tested. Discuss material API, wire-contract, ledger, eligibility, or fulfillment changes before implementation. Keep the Swift package dependency-free, preserve strict concurrency, add focused tests, and run the Swift, Python, and Worker verification commands in the README.

Never commit secrets, generated Apple offer codes, local setup state, `.dev.vars`, Wrangler state, or production database identifiers.
